How to Use

Strategic Communications encourages the use of SVGs. All icons are available in SVG format and can be used as raw SVG code or linking to through an image.

Use as an image

<!-- Download -->
<img alt="download" src="/img/icons/download-black.svg">
<img alt="download" src="/img/icons/download-white.svg">
<img alt="download" src="/img/icons/download-blue.svg">
